Translate WordPress plugins and themes directly in your browser


== Description ==

Plugin to detect hyperlinks, links, special characters, avoid code injection in your forms.

It also generates and displays an alert message to provide the user with real-time feedback.


== Installation ==

= Basic usage: =

1. Download the Plugin from the Official WordPress Repository.
2. Activate The Plugin From Your WordPress Administration Panel.
3. Test the functionality, trying to enter some URL in your contact forms.



= Installing manually: =

1. Unzip the entire "secure-forms" folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the **Plugins** screen (**Plugins > Installed Plugins**).
